## 什么是CVF
- CVF(CuberVisualFramework)是基于Zepp OS API的开源UI框架。
## CVF的特点 
- 将屏幕空间由二维扩展至三维，并通过3D视角模拟实现等立体的UI布局
- 组件、渲染与控制分离，扩展性强
- 高优化、低负载，常见情况达到60FPS
- 内置各种动画、控件与布局管理器
